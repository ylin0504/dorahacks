# Glossary

A short list of terms commonly used on DoraHacks.

### BUIDL

A project created and maintained by builders. On DoraHacks every project is represented by a BUIDL page with description, links, and team information.

### Hackathon

A time bounded event where developers build prototypes, submit BUIDLs, and compete for prizes or ecosystem support.

### Grant Round

A funding program where projects apply for support. Often uses quadratic funding or other mechanisms to allocate resources.

### Quadratic Funding (QF)

A funding mechanism that matches contributions from the community and a matching pool. It favors projects that receive support from many people instead of only a few large donors.

### MACI

Minimal Anti Collusion Infrastructure. A system that uses cryptography to help prevent vote buying and coercion in voting and funding rounds.

### Dora Factory

The infrastructure layer that provides governance and funding tools such as QF, MACI, and other modules used by DoraHacks and ecosystems.

### Hacker Profile

The user profile for builders and developers on DoraHacks.

### Organization Profile

The profile for ecosystems, foundations, and organizations that host events or programs on DoraHacks.
