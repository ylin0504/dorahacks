# Mission and Vision

DoraHacks is on a mission to connect builders, capital, and ecosystems so that important technologies and public goods can be created faster and adopted by the world.

The platform helps developers around the globe:

- Find like minded teammates  
- Validate ideas through hackathons and grants  
- Access long term funding and ecosystem support  

DoraHacks aims to be the infrastructure layer for global hacker movements and developer ecosystems.
